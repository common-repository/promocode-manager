Promocode Manager lets user define products, partners, and promotions to generate shortcodes that can be placed anywhere in your site. Once a user has validated against your promotion form, new prices/attributes will update using javascript.

== Description ==

Promocode Manager is a wordpress plugin used to setup products, partners, and promotions on your posts/pages.

Shortcodes are setup to be placed on your post/pages that will display predefined product/partner/promo data defined in the backend.

== Installation ==

1. Upload the Promocode plugin folder into your blog's 'wp-content/plugins' directory.
2. Activate it through the wordpress admin panel, an admin nav item will display on successful activation.
3. After creating partners, products, and promos, drop any plugin specific shortcodes to get started (refer to the help section for more information)
